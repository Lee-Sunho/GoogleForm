## **💡 실행 방법**

- git clone
- npm i
- npm start

- 또는 배포 링크 https://lee-sunho.github.io/googleform/ 를 통해 실행할 수 있습니다.

## **🔨 사용 기술 스택**

- react: ^18.2.0
- typescript: ^4.9.5
- react-redux: ^8.1.1
- reduxjs/toolkit: ^1.9.5
- redux-persist: ^6.0.0
- styled-components: ^6.0.5
- react-beautiful-dnd: ^13.1.1
- react-router-dom: ^6.14.2
